# AJJRAM BRASS — Luxury Heritage Brass E-Commerce Platform

> **A Complete, Premium E-Commerce Experience with Amazon/Flipkart-Level Features + Luxury Heritage Branding**

---

## ✨ OVERVIEW

Ajjram Brass is a **full-featured luxury e-commerce platform** purpose-built for a premium heritage brass brand. It combines the comprehensive functionality of major marketplaces (Amazon, Flipkart, Myntra) with the exclusive feel of a luxury heritage brand.

The platform includes a **complete customer shopping experience**, a **full seller dashboard**, and a **custom brass ordering system** — all wrapped in a stunning premium design with a Deep Heritage Green & Antique Gold color palette.

## 🔄 KEY WORKFLOW: CUSTOMER ORDER → FULFILLMENT

```
CUSTOMER                          SELLER
─────────                        ──────
   │                                │
   ├─ Browse Products ──────────────┤
   ├─ Add to Cart                   │
   ├─ Checkout                      │
   │  ├─ Select Address             │
   │  ├─ Choose Payment (UPI/Card/Net Banking/COD)
   │  └─ Place Order                │
   │                                │
   │                          ╔══════════════╗
   │                          ║  ORDER PLACED ║
   │                          ╚══════════════╝
   │                                │
   │                                ├─ Seller confirms order
   │                                │  → Status: CONFIRMED
   │                                ├─ Seller packs product
   │                                │  → Status: PACKED
   │                                ├─ Seller ships (enters Shiprocket ID)
   │                                │  → Status: SHIPPED
   │                                │
   ├─ ═══════════ IN TRANSIT ═══════┤
   │                                │
   ├─ Out for Delivery              │
   │  → Status: OUT FOR DELIVERY    │
   ├─ Delivered                     │
   │  → Status: DELIVERED           │
   │                                │
   ├─ Download Invoice              │
   ├─ Leave Review                  │
   └─ Track via Timeline ───────────┘
```
